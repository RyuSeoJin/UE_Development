RecordTales의 7개 서브시스템을 UMG 위젯으로 연결하는 실제 구현 절차입니다. UIManagerSubsystem의 패널/모달 관리 로직부터 개별 위젯의 C++ 클래스·Designer 레이아웃·델리게이트 바인딩까지, 따라 하면 UI가 완성됩니다. UMG 기초 지식은 「UMG + C++ 연동 프레임워크」 문서를 전제합니다.

02 — 설계 결정

설계 결정

책임 배치 결정표

각 기능을 어느 클래스에 둘 것인지, 그 근거를 명시합니다.

이 기능둘 곳근거
패널 전환 로직 (한 번에 하나) UUIManagerSubsystem 레벨 전환에도 유지, 모든 위젯 라이프사이클 일괄 관리
모달 스택 (Push/Pop) UUIManagerSubsystem 여러 모달의 순서 관리는 위젯 자신이 할 수 없음
토스트 큐 UUIManagerSubsystem 복수 토스트 순차 표시 정책
타이머 표시 갱신 UFocusPanelWidget 표시 로직은 해당 위젯에, 데이터는 FocusSubsystem에서 받음
퀘스트 목록 렌더링 UQuestPanelWidget ScrollBox 관리는 위젯 담당
가챠 연출 UGachaModalWidget 시각 연출은 위젯 BP의 UWidgetAnimation 활용
위젯 인스턴스 소유 UUIManagerSubsystem UPROPERTY()로 참조 보유 → GC 방지

UIManagerSubsystem 위젯 관리 패턴

서브시스템이 모든 위젯 인스턴스를 UPROPERTY() TMap<FName, UUserWidget*>으로 소유합니다. 핵심 흐름:

  • ShowPanel() → 기존 패널 RemoveFromParent() → 새 패널 AddToViewport(5)
  • ShowModal() → 모달 Push (스택) → AddToViewport(10 + 스택깊이)
  • HideModal() → Pop → RemoveFromParent()

Widget Blueprint 클래스 할당 방식

ℹ

DataAsset 방식 권장
UIManagerSubsystem에 UPROPERTY(EditDefaultsOnly) TSubclassOf<UFocusPanelWidget> FocusPanelClass; 를 직접 두는 방식도 가능하지만, URecordTalesUIConfig(DataAsset)으로 분리하여 7개 WBP 경로를 한 곳에서 관리하는 것을 권장합니다. 에디터에서 DA_UIConfig 에셋 하나만 열면 모든 위젯 클래스 할당을 한 눈에 확인/편집할 수 있습니다.

04 — 파일별 코드: 포커스 패널

포커스 패널 구현

한 위젯을 처음부터 끝까지 완전히 만드는 예제입니다. 이 패턴을 숙지하면 나머지 위젯도 동일한 방식으로 만들 수 있습니다.

공통 베이스 클래스 .h .cpp

모든 RecordTales 위젯이 상속하는 베이스 클래스입니다. NativeConstruct()에서 7개 서브시스템을 캐싱해 자식 위젯에서 바로 사용합니다.

#pragma once
#include "CoreMinimal.h"
#include "Blueprint/UserWidget.h"
#include "RecordTalesWidgetBase.generated.h"

class UFocusSubsystem;
class UQuestSubsystem;
class URecordSubsystem;
class UEconomySubsystem;
class UGuildSubsystem;
class UAudioSubsystem;
class UUIManagerSubsystem;

UCLASS(Abstract, Blueprintable)
class RECORDTALES_API URecordTalesWidgetBase : public UUserWidget
{
    GENERATED_BODY()
protected:
    virtual void NativeConstruct() override;

    // 서브시스템 캐시 — 모든 위젯에서 사용 가능
    UPROPERTY()
    UFocusSubsystem* FocusSub;
    UPROPERTY()
    UEconomySubsystem* EconomySub;
    UPROPERTY()
    UQuestSubsystem* QuestSub;
    UPROPERTY()
    URecordSubsystem* RecordSub;
    UPROPERTY()
    UGuildSubsystem* GuildSub;
    UPROPERTY()
    UAudioSubsystem* AudioSub;
    UPROPERTY()
    UUIManagerSubsystem* UIMgrSub;
};
#include "Widgets/RecordTalesWidgetBase.h"
#include "Focus/FocusSubsystem.h"
#include "Economy/EconomySubsystem.h"
#include "Quest/QuestSubsystem.h"
#include "Record/RecordSubsystem.h"
#include "Guild/GuildSubsystem.h"
#include "Audio/AudioSubsystem.h"
#include "UI/UIManagerSubsystem.h"

void URecordTalesWidgetBase::NativeConstruct()
{
    Super::NativeConstruct();

    UGameInstance* GI = GetGameInstance();
    FocusSub   = GI->GetSubsystem<UFocusSubsystem>();
    EconomySub = GI->GetSubsystem<UEconomySubsystem>();
    QuestSub   = GI->GetSubsystem<UQuestSubsystem>();
    RecordSub  = GI->GetSubsystem<URecordSubsystem>();
    GuildSub   = GI->GetSubsystem<UGuildSubsystem>();
    AudioSub   = GI->GetSubsystem<UAudioSubsystem>();
    UIMgrSub   = GI->GetSubsystem<UUIManagerSubsystem>();
}

FocusPanelWidget .h .cpp

포커스 시스템의 타이머·상태·Gold 수익을 실시간으로 표시하는 핵심 패널입니다. BindWidget 매크로로 Designer와 연결하고, NativeConstruct에서 버튼·델리게이트를 바인딩합니다.

#pragma once
#include "Widgets/RecordTalesWidgetBase.h"
#include "FocusPanelWidget.generated.h"

class UTextBlock;
class UButton;
class UProgressBar;

UCLASS(Blueprintable)
class RECORDTALES_API UFocusPanelWidget : public URecordTalesWidgetBase
{
    GENERATED_BODY()

public:
    // BindWidget — Designer에서 이 이름으로 위젯 배치 필수
    UPROPERTY(meta=(BindWidget))
    UTextBlock* TXT_TimerDisplay;        // "24:59" 형태 표시

    UPROPERTY(meta=(BindWidget))
    UTextBlock* TXT_Mode;                // "집중 중" / "휴식 중" / "대기"

    UPROPERTY(meta=(BindWidget))
    UTextBlock* TXT_CycleCount;          // "2 / 4 사이클"

    UPROPERTY(meta=(BindWidget))
    UTextBlock* TXT_GoldEarned;          // "+1,234 Gold"

    UPROPERTY(meta=(BindWidget))
    UButton* BTN_Start;

    UPROPERTY(meta=(BindWidget))
    UButton* BTN_Pause;

    UPROPERTY(meta=(BindWidget))
    UButton* BTN_Stop;

    UPROPERTY(meta=(BindWidget))
    UProgressBar* PB_Timer;              // 남은 시간 비율

    UPROPERTY(meta=(BindWidgetOptional))
    UTextBlock* TXT_XPEarned;            // 선택: XP 표시

protected:
    virtual void NativeConstruct() override;
    virtual void NativeDestruct() override;
    virtual void NativeTick(const FGeometry& MyGeometry, float InDeltaTime) override;

private:
    UFUNCTION()
    void OnStartClicked();
    UFUNCTION()
    void OnPauseClicked();
    UFUNCTION()
    void OnStopClicked();

    void HandleStateChanged(EFocusState NewState);
    void HandleGoldTick(int32 TotalGold);
    void UpdateTimerDisplay();

    int32 SessionGold = 0;
};
#include "Widgets/FocusPanelWidget.h"
#include "Components/TextBlock.h"
#include "Components/Button.h"
#include "Components/ProgressBar.h"
#include "Focus/FocusSubsystem.h"
#include "Economy/EconomySubsystem.h"

void UFocusPanelWidget::NativeConstruct()
{
    Super::NativeConstruct();  // 서브시스템 캐싱 완료

    // 버튼 바인딩
    BTN_Start->OnClicked.AddDynamic(this, &UFocusPanelWidget::OnStartClicked);
    BTN_Pause->OnClicked.AddDynamic(this, &UFocusPanelWidget::OnPauseClicked);
    BTN_Stop->OnClicked.AddDynamic(this, &UFocusPanelWidget::OnStopClicked);

    // 델리게이트 바인딩
    FocusSub->OnStateChanged.AddUObject(this, &UFocusPanelWidget::HandleStateChanged);
    EconomySub->OnGoldChanged.AddUObject(this, &UFocusPanelWidget::HandleGoldTick);

    // 초기 상태 반영
    HandleStateChanged(FocusSub->GetCurrentState());
    SessionGold = 0;
}

void UFocusPanelWidget::NativeDestruct()
{
    if (FocusSub) FocusSub->OnStateChanged.RemoveAll(this);
    if (EconomySub) EconomySub->OnGoldChanged.RemoveAll(this);
    BTN_Start->OnClicked.RemoveAll(this);
    BTN_Pause->OnClicked.RemoveAll(this);
    BTN_Stop->OnClicked.RemoveAll(this);
    Super::NativeDestruct();
}

void UFocusPanelWidget::NativeTick(const FGeometry& MyGeometry, float InDeltaTime)
{
    Super::NativeTick(MyGeometry, InDeltaTime);
    if (FocusSub->GetCurrentState() == EFocusState::Focusing ||
        FocusSub->GetCurrentState() == EFocusState::Resting)
    {
        UpdateTimerDisplay();
    }
}

void UFocusPanelWidget::OnStartClicked()
{
    FocusSub->StartFocus();
    SessionGold = 0;
}

void UFocusPanelWidget::OnPauseClicked()
{
    FocusSub->PauseTimer();
}

void UFocusPanelWidget::OnStopClicked()
{
    FocusSub->StopFocus();
}

void UFocusPanelWidget::HandleStateChanged(EFocusState NewState)
{
    switch (NewState)
    {
    case EFocusState::Idle:
        TXT_Mode->SetText(FText::FromString(TEXT("대기")));
        BTN_Start->SetVisibility(ESlateVisibility::Visible);
        BTN_Pause->SetVisibility(ESlateVisibility::Collapsed);
        BTN_Stop->SetVisibility(ESlateVisibility::Collapsed);
        break;
    case EFocusState::Focusing:
        TXT_Mode->SetText(FText::FromString(TEXT("집중 중")));
        BTN_Start->SetVisibility(ESlateVisibility::Collapsed);
        BTN_Pause->SetVisibility(ESlateVisibility::Visible);
        BTN_Stop->SetVisibility(ESlateVisibility::Visible);
        break;
    case EFocusState::Resting:
        TXT_Mode->SetText(FText::FromString(TEXT("휴식 중")));
        break;
    case EFocusState::Complete:
        TXT_Mode->SetText(FText::FromString(TEXT("완료!")));
        BTN_Start->SetVisibility(ESlateVisibility::Visible);
        BTN_Pause->SetVisibility(ESlateVisibility::Collapsed);
        BTN_Stop->SetVisibility(ESlateVisibility::Collapsed);
        break;
    }
}

void UFocusPanelWidget::HandleGoldTick(int32 TotalGold)
{
    SessionGold++;
    TXT_GoldEarned->SetText(FText::Format(
        NSLOCTEXT("Focus", "GoldFmt", "+{0} Gold"),
        FText::AsNumber(SessionGold)));
}

void UFocusPanelWidget::UpdateTimerDisplay()
{
    float Remaining = FocusSub->GetRemainingTime();
    int32 Minutes = FMath::FloorToInt(Remaining / 60.f);
    int32 Seconds = FMath::FloorToInt(FMath::Fmod(Remaining, 60.f));
    TXT_TimerDisplay->SetText(FText::FromString(
        FString::Printf(TEXT("%02d:%02d"), Minutes, Seconds)));

    float Total = FocusSub->GetTotalDuration();
    if (Total > 0) PB_Timer->SetPercent(Remaining / Total);
}

• BTN_Stop (Button) — Pause 옆
• PB_Timer (ProgressBar) — 타이머 아래 전체 너비
• TXT_XPEarned (TextBlock, optional) — 좌측 상단
5
컴파일 & 저장
Compile › Save
⚠
위젯 이름에 공백이나 한글 사용 금지! C++ 변수명과 정확히 일치해야 합니다. 대소문자까지 동일해야 BindWidget이 매칭합니다.
⚠

05 — 파일별 코드: 퀘스트 패널

퀘스트 패널 구현

ScrollBox에 동적으로 자식 위젯을 추가하는 동적 리스트 패턴의 대표 예제입니다. 이 패턴은 아카이브, 길드 캐릭터 그리드 등 모든 목록 UI에 재사용됩니다.

QuestPanelWidget.h (핵심 부분)

#pragma once
#include "Widgets/RecordTalesWidgetBase.h"
#include "QuestPanelWidget.generated.h"

class UScrollBox;
class UButton;
class UComboBoxString;
class UQuestEntryWidget;

UCLASS(Blueprintable)
class RECORDTALES_API UQuestPanelWidget : public URecordTalesWidgetBase
{
    GENERATED_BODY()

public:
    UPROPERTY(meta=(BindWidget))
    UScrollBox* SB_QuestList;

    UPROPERTY(meta=(BindWidget))
    UButton* BTN_AddQuest;

    UPROPERTY(meta=(BindWidget))
    UComboBoxString* CB_Filter;  // 상태 필터 (전체/진행중/완료)

    UPROPERTY(EditDefaultsOnly)
    TSubclassOf<UQuestEntryWidget> QuestEntryClass;

protected:
    virtual void NativeConstruct() override;

private:
    void RefreshQuestList();
    UFUNCTION()
    void OnAddQuestClicked();
    UFUNCTION()
    void OnFilterChanged(FString SelectedItem, ESelectInfo::Type SelectionType);
};
#include "Widgets/QuestPanelWidget.h"
#include "Components/ScrollBox.h"
#include "Components/Button.h"
#include "Components/ComboBoxString.h"
#include "Components/QuestEntryWidget.h"
#include "Quest/QuestSubsystem.h"

void UQuestPanelWidget::NativeConstruct()
{
    Super::NativeConstruct();

    BTN_AddQuest->OnClicked.AddDynamic(this, &UQuestPanelWidget::OnAddQuestClicked);
    CB_Filter->OnSelectionChanged.AddDynamic(this, &UQuestPanelWidget::OnFilterChanged);

    // 퀘스트 변경 시 자동 갱신
    QuestSub->OnQuestListChanged.AddUObject(this, &UQuestPanelWidget::RefreshQuestList);

    RefreshQuestList();
}

void UQuestPanelWidget::RefreshQuestList()
{
    // 1. 기존 자식 위젯 모두 제거
    SB_QuestList->ClearChildren();

    // 2. 서브시스템에서 퀘스트 목록 조회
    TArray<FQuestMasterData> Quests = QuestSub->GetAllQuests();

    // 3. 각 퀘스트에 대해 엔트리 위젯 생성 후 ScrollBox에 추가
    for (const FQuestMasterData& Quest : Quests)
    {
        UQuestEntryWidget* Entry = CreateWidget<UQuestEntryWidget>(this, QuestEntryClass);
        Entry->SetQuestData(Quest);
        SB_QuestList->AddChild(Entry);
    }
}

void UQuestPanelWidget::OnAddQuestClicked()
{
    // 퀘스트 추가 UI 로직 (입력 모달 등)
    QuestSub->AddNewQuest(TEXT("새 퀘스트"));
}

void UQuestPanelWidget::OnFilterChanged(FString SelectedItem, ESelectInfo::Type SelectionType)
{
    // 필터에 따라 목록 갱신
    RefreshQuestList();
}

ℹ

ScrollBox 동적 리스트 패턴 요약: ClearChildren() → 데이터 루프 → CreateWidget<T>(this, Class) → SetData() → AddChild(). 이 패턴은 캐릭터 그리드, 기록 리스트 등 모든 목록 UI에 그대로 재사용됩니다.

⚠
ClearChildren() 후 새로 CreateWidget하면 이전 위젯은 GC 수거됩니다. 수백 개 아이템이 있는 리스트라면 오브젝트 풀링(Pool)을 고려하세요. RecordTales의 퀘스트 수는 보통 20개 미만이므로 여기서는 불필요합니다.

#pragma once
#include "Widgets/RecordTalesWidgetBase.h"
#include "RecordTalesHUD.generated.h"

class UCanvasPanel;
class UVerticalBox;
class UHorizontalBox;
class UNamedSlot;

UCLASS(Blueprintable)
class RECORDTALES_API URecordTalesHUD : public URecordTalesWidgetBase
{
    GENERATED_BODY()

public:
    UPROPERTY(meta=(BindWidget))
    UCanvasPanel* Canvas_Root;

    // 4코너
    UPROPERTY(meta=(BindWidget))
    UVerticalBox* VB_TopLeft;       // 길드 레벨 + XP 바

    UPROPERTY(meta=(BindWidget))
    UHorizontalBox* HB_TopRight;    // Gold + 설정 아이콘

    UPROPERTY(meta=(BindWidget))
    UVerticalBox* VB_BottomLeft;    // 앰비언트/뮤직 미니플레이어

    UPROPERTY(meta=(BindWidget))
    UVerticalBox* VB_BottomRight;   // 토스트 영역

    // 중앙 패널 영역
    UPROPERTY(meta=(BindWidget))
    UNamedSlot* NS_CenterPanel;     // 패널 위젯이 교체되는 슬롯
};

NamedSlot 패턴

NamedSlot은 Designer에서 "다른 위젯을 끼워 넣을 수 있는 빈 칸"입니다. UIManagerSubsystem이 패널 전환 시 이 슬롯의 Content를 교체합니다. 또는 단순히 AddToViewport로 패널을 올리고 HUD는 항상 위에 있는 방식(ZOrder 활용)도 가능합니다.

Designer 앵커 설정

위젯앵커위치
VB_TopLeft좌상 (0,0)Offset: (20, 20)
HB_TopRight우상 (1,0)Alignment: (1,0), Offset: (-20, 20)
VB_BottomLeft좌하 (0,1)Alignment: (0,1), Offset: (20, -20)
VB_BottomRight우하 (1,1)Alignment: (1,1), Offset: (-20, -20)
NS_CenterPanel중앙 스트레치Anchors: (0.15, 0.1)~(0.85, 0.9)
✓

앵커 설정 핵심 원리: 앵커는 "부모의 어디를 기준으로 이 위젯의 위치를 잡을까"를 정합니다. 좌상(0,0)에 앵커를 두면 부모의 왼쪽 위 꼭짓점으로부터 Offset 만큼 떨어진 곳에 배치됩니다. 창 크기가 바뀌어도 4코너의 앵커가 유지되므로 레이아웃이 깨지지 않습니다.

07 — 엔진 등록

엔진 등록

UIManagerSubsystem에서 HUD 생성

1
UIManagerSubsystem::Initialize()에서 HUD 위젯 생성
CreateWidget<URecordTalesHUD>(GetWorld(), HUDWidgetClass) 호출. 단, Initialize() 시점에 World가 없을 수 있으므로 첫 Tick 또는 UGameInstanceSubsystem::Initialize() 후 World 로드 이벤트에서 처리합니다.
2
AddToViewport(0)
HUD는 ZOrder 0 (가장 아래). 패널은 ZOrder 5, 모달은 ZOrder 10+.
3
패널/모달은 ShowPanel()/ShowModal()에서 동적 생성
이미 인스턴스가 있으면 재사용, 없으면 CreateWidget → AddToViewport(ZOrder).

DataAsset 설정

1
DataAsset 생성
Content Browser › 우클릭 › Miscellaneous › Data Asset › URecordTalesUIConfig 선택 › 이름: DA_UIConfig
2
WBP 할당
DA_UIConfig 열기 › 각 위젯 클래스 슬롯에 해당 WBP 할당 (WBP_FocusPanel, WBP_QuestPanel 등 7개)
3
서브시스템에서 참조
UUIManagerSubsystem의 Initialize()에서 DA_UIConfig를 로드하여 각 TSubclassOf를 가져옵니다. TSoftObjectPtr로 경로 지정 또는 GameInstance 세팅에서 직접 참조합니다.
