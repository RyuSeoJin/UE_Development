01 — 개념 이해

포커스 시스템이란?

포커스 시스템은 포모도로(Pomodoro) 기법(25분 집중 + 5분 휴식을 반복하는 시간 관리 방법) 기반의 집중 타이머와, 정해진 시간 없이 0초부터 자유롭게 측정하는 스톱워치를 하나의 서브시스템(UFocusSubsystem)으로 통합한 것입니다.

사용자는 두 모드 중 하나를 선택해 집중 세션을 시작합니다. 타이머 모드에서는 설정된 시간이 카운트다운되고, 스톱워치 모드에서는 경과 시간이 카운트업됩니다. 세션이 완료되면 XP(경험치)와 Gold(게임 내 화폐)가 적립되어, 길드원 모집과 레벨업에 사용됩니다.

핵심 용어

📖

포모도로(Pomodoro) — 25분 집중 + 5분 휴식을 1사이클로 하는 시간 관리 기법. "토마토"라는 뜻의 이탈리아어에서 유래.
루틴(Routine) — 포모도로 사이클을 연속으로 반복하는 것. 1회면 단일 모드, 2회 이상이면 루틴 모드.
세션(Session) — 시작 버튼을 누른 순간부터 모든 루틴이 끝나는 순간까지의 전체 시간. 하나의 세션에는 여러 사이클이 포함될 수 있음.
사이클(Cycle) — 집중 1회 + 휴식 1회를 묶은 단위. 루틴 4회 = 4사이클.
FocusSubsystem — UGameInstanceSubsystem을 상속한 서브시스템. 타이머/스톱워치 상태를 관리하고, 보상 적립과 기록 전달을 트리거함.

02 — 타이머 상태 머신

5가지 상태와 전환 규칙

타이머는 상태 머신(State Machine)으로 동작합니다. 상태 머신이란 "현재 어떤 상태에 있는지"와 "어떤 조건이 충족되면 다음 상태로 넘어가는지"를 명확히 정의한 설계 패턴입니다. 포커스 타이머의 상태는 5가지이며, 각 상태에서 할 수 있는 행동과 UI 표시가 다릅니다.

상태 전환 흐름

대기
Idle
설정 UI 표시. 시작 버튼 대기.
→
집중
Focusing
카운트다운 링. Gold 초당 +1.
→
휴식
Resting / LastResting
휴식 카운트다운. Gold 멈춤.

상태 상세 테이블

상태의미전환 조건UI 표시
Idle 대기 중. 아직 집중을 시작하지 않은 초기 상태 시작 버튼 클릭 → Focusing 타이머 설정 UI (시간/루틴 조절)
Focusing 집중 중. 설정한 시간(기본 25분)이 카운트다운됨 시간 만료 → Resting 또는 LastResting 카운트다운 링 + 남은 시간 표시
Resting 일반 휴식. 루틴의 중간 사이클에서 집중 후 쉬는 시간 시간 만료 → Focusing (다음 사이클) 휴식 카운트다운 + 현재/전체 사이클 표시
LastResting 마지막 휴식. 루틴의 마지막 사이클에서만 발생. 보통 더 긴 휴식(기본 15분) 시간 만료 → Idle (또는 자동 재시작 시 Focusing) 마지막 휴식 카운트다운 + 완료 안내
Paused 일시정지. 어떤 상태에서든 일시정지 가능. 남은 시간이 보존됨 재개 → 이전 상태(Focusing 또는 Resting 등) 일시정지 아이콘 + 남은 시간 고정 표시

단일 모드 vs 루틴 모드

단일 모드 (루틴 1회)

반복 없이 1사이클만 진행합니다. 집중이 끝나면 단일 휴식 후 Idle로 돌아갑니다. 짧은 작업이나 시험 용도에 적합합니다.

Focusing Resting Idle

루틴 모드 (2회 이상)

설정한 횟수만큼 [집중 → 휴식]을 반복합니다. 마지막 사이클의 휴식만 LastResting(긴 휴식)으로 전환됩니다.

Focusing Resting Focusing Resting Focusing LastResting Idle
💡

자동 재시작이 켜지면 LastResting 후 Idle 대신 Focusing으로 복귀합니다. 사이클 인덱스는 1로 초기화되어 새 세션이 시작됩니다. 자동 재시작은 장시간 반복 작업(공부 마라톤 등)에 유용합니다.

03 — 타이머 설정

사용자가 조절할 수 있는 값

타이머 설정은 Idle 상태에서만 변경할 수 있습니다. 설정값은 URecordTalesSaveGame에 저장되어 앱을 재시작해도 유지됩니다.

설정기본값범위저장 위치
집중 시간 (FocusDuration)25분 (1500초)1 ~ 120분SaveGame
일반 휴식 (RestDuration)5분 (300초)1 ~ 30분SaveGame
마지막 휴식 (LastRestDuration)15분 (900초)1 ~ 60분SaveGame
반복 횟수 (RoutineCount)4회1 ~ 10회SaveGame
자동 재시작 (bAutoRestart)OffOn / OffSaveGame

설정 구조체

FFocusSettings (USTRUCT)
USTRUCT(BlueprintType)
struct FFocusSettings
{
    GENERATED_BODY()

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float FocusDuration = 1500.f;    // 25분 (초)

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float RestDuration = 300.f;      // 5분 (초)

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float LastRestDuration = 900.f;  // 15분 (초)

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    int32 RoutineCount = 4;          // 반복 횟수

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    bool bAutoRestart = false;       // 자동 재시작
};
04 — 스톱워치

자유 시간 측정 모드

스톱워치 모드는 정해진 시간 없이 0초부터 경과 시간을 측정합니다. 타이머와 달리 카운트다운이 아닌 카운트업이며, 사용자가 직접 중지 버튼을 눌러야 세션이 종료됩니다. 코딩, 독서 등 "얼마나 했는지"를 추적하고 싶을 때 사용합니다.

스톱워치 상태

상태의미전환 조건
Idle아직 시작하지 않음. 경과 시간 0초시작 버튼 → Running
Running시간 측정 중. 매 프레임 경과 시간 누적일시정지 → Paused, 중지 → Idle (기록 저장)
Paused일시정지. 경과 시간 고정재개 → Running, 중지 → Idle (기록 저장)

XP 적립 자격

스톱워치 모드는 포모도로와 다른 보상 규칙을 적용합니다. 최소 25분(1500초) 이상 경과해야 XP 적립 자격이 생기며, 분당 +10 XP로 계산합니다. 이 최소 시간 제한은 짧은 시간 남용(10초 켜고 끄기)을 방지합니다.

⚠️

스톱워치는 포모도로 보상(+15 XP)을 받지 않습니다. 포모도로 보상은 정해진 시간(기본 25분)을 온전히 집중해야 받는 보상이고, 스톱워치는 자유 시간이므로 분당 +10 XP로 별도 계산합니다. Gold는 동일하게 초당 +1이 적용됩니다.

05 — XP/Gold 보상 규칙

집중이 보상으로 이어지는 경로

포커스 시스템의 핵심 동기부여는 집중할수록 Gold와 XP가 쌓이는 것입니다. Gold(게임 내 화폐)는 길드원 가챠(1000 Gold)에 쓰이고, XP(경험치)는 길드 레벨을 올려 더 좋은 가챠 확률과 해금을 제공합니다. 아래 표는 모든 보상 규칙을 정리합니다.

행동보상조건
포모도로 1사이클 완료 +15 XP 집중 시간이 만료될 때까지 완료해야 함. 중도 중지 시 미지급.
연속 3사이클 완료 +20 XP 보너스 한 세션에서 3회 연속 사이클을 완료. 4회, 5회 등에서도 3의 배수마다 추가 지급.
스톱워치 XP 분당 +10 XP 경과 시간이 1500초(25분) 이상일 때만 적립. 소수점 이하 버림.
집중 중 Gold 틱 초당 +1 Gold Focusing 상태(타이머)이거나 Running 상태(스톱워치)에서만 틱. 휴식/일시정지 시 멈춤.
퀘스트 완료 Gold +200 Gold 퀘스트 1건 완료 시. 일일 상한 2000 Gold (10건). QuestSubsystem 관할.

보상 적립 흐름

1
상태 변경 감지
UFocusSubsystem에서 사이클 완료(Focusing 만료) 또는 스톱워치 중지 이벤트가 발생합니다. 이 시점에 적립 자격을 판단합니다.
2
EconomySubsystem 호출
UEconomySubsystem::AddXP(Amount)과 AddGold(Amount)를 호출해 런타임 잔고를 갱신합니다. 델리게이트(앞서 설명한 콜백 시스템)를 통해 호출하면 결합도가 낮아집니다.
3
레벨업 조건 확인
UEconomySubsystem 내부에서 누적 XP가 다음 레벨 문턱을 넘는지 확인합니다. 넘으면 레벨을 올리고 OnLevelUp 델리게이트를 브로드캐스트합니다.
4
SaveGame에 기록
갱신된 Gold/XP/레벨 값을 URecordTalesSaveGame에 복사하고 디스크에 저장합니다. 동시에 URecordSubsystem에 세션 기록(시작 시각, 종료 시각, 집중 시간, 획득 보상)을 전달합니다.
🧭

Gold 틱은 Focusing/Running 상태에서만 작동합니다. Resting, LastResting, Paused 상태에서는 Gold 틱이 멈춥니다. 이는 "집중한 시간만큼만 보상한다"는 설계 원칙을 반영합니다.

06 — 구현 스켈레톤

UFocusSubsystem 최소 코드

아래는 UFocusSubsystem의 핵심 뼈대입니다. 상태 전환, 타이머 설정, 주요 함수 시그니처를 포함하며, 세부 구현(UI 바인딩, 사운드 연동 등)은 프로젝트에 맞게 확장합니다. 헤더(FocusSubsystem.h)에 선언하고, 구현은 소스(FocusSubsystem.cpp)에 둡니다. 

#pragma once

#include "CoreMinimal.h"
#include "Subsystems/GameInstanceSubsystem.h"
#include "FocusSubsystem.generated.h"

// ── 타이머 상태 열거형 ──────────────────────────────
UENUM(BlueprintType)
enum class EFocusState : uint8
{
    Idle          UMETA(DisplayName = "Idle"),
    Focusing      UMETA(DisplayName = "Focusing"),
    Resting       UMETA(DisplayName = "Resting"),
    LastResting   UMETA(DisplayName = "LastResting"),
    Paused        UMETA(DisplayName = "Paused")
};

// ── 스톱워치 상태 열거형 ────────────────────────────
UENUM(BlueprintType)
enum class EStopwatchState : uint8
{
    Idle,
    Running,
    Paused
};

// ── 타이머 설정 구조체 ──────────────────────────────
USTRUCT(BlueprintType)
struct FFocusSettings
{
    GENERATED_BODY()

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float FocusDuration = 1500.f;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float RestDuration = 300.f;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float LastRestDuration = 900.f;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    int32 RoutineCount = 4;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    bool bAutoRestart = false;
};

// ── 사이클 완료 델리게이트 ──────────────────────────
DECLARE_DYNAMIC_MULTICAST_DELEGATE_TwoParams(
    FOnFocusCycleCompleted, int32, CycleIndex, float, FocusDuration);
DECLARE_DYNAMIC_MULTICAST_DELEGATE_OneParam(
    FOnStopwatchStopped, float, ElapsedSeconds);

// ── UFocusSubsystem ─────────────────────────────────
UCLASS()
class RECORDTALES_API UFocusSubsystem : public UGameInstanceSubsystem
{
    GENERATED_BODY()

public:
    // --- 초기화 ---
    virtual void Initialize(FSubsystemCollectionBase& Collection) override;
    virtual void Deinitialize() override;

    // --- 타이머 제어 ---
    UFUNCTION(BlueprintCallable, Category = "Focus")
    void StartFocus();

    UFUNCTION(BlueprintCallable, Category = "Focus")
    void PauseFocus();

    UFUNCTION(BlueprintCallable, Category = "Focus")
    void ResumeFocus();

    UFUNCTION(BlueprintCallable, Category = "Focus")
    void StopFocus();

    // --- 스톱워치 제어 ---
    UFUNCTION(BlueprintCallable, Category = "Focus")
    void StartStopwatch();

    UFUNCTION(BlueprintCallable, Category = "Focus")
    void StopStopwatch();

    // --- 상태 조회 ---
    UFUNCTION(BlueprintPure, Category = "Focus")
    EFocusState GetFocusState() const { return CurrentState; }

    UFUNCTION(BlueprintPure, Category = "Focus")
    float GetRemainingTime() const { return RemainingTime; }

    UFUNCTION(BlueprintPure, Category = "Focus")
    int32 GetCurrentCycle() const { return CurrentCycleIndex; }

    // --- 델리게이트 ---
    UPROPERTY(BlueprintAssignable)
    FOnFocusCycleCompleted OnFocusCycleCompleted;

    UPROPERTY(BlueprintAssignable)
    FOnStopwatchStopped OnStopwatchStopped;

    // --- 설정 ---
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Focus")
    FFocusSettings Settings;

private:
    void OnTimerExpired();          // FTimerHandle 콜백
    void TransitionTo(EFocusState NewState);
    void BeginTimerForState(EFocusState State);

    EFocusState CurrentState = EFocusState::Idle;
    EFocusState StateBeforePause = EFocusState::Idle;
    EStopwatchState SWState = EStopwatchState::Idle;

    FTimerHandle FocusTimerHandle;
    float RemainingTime = 0.f;     // 일시정지 시 남은 시간 보존용
    float StopwatchElapsed = 0.f;  // 스톱워치 경과 시간

    int32 CurrentCycleIndex = 0;   // 현재 사이클 (0-based)
    int32 ConsecutiveCount = 0;    // 연속 완료 카운트 (보너스 XP용)
};
#include "FocusSubsystem.h"
#include "Engine/World.h"
#include "TimerManager.h"

void UFocusSubsystem::Initialize(FSubsystemCollectionBase& Collection)
{
    Super::Initialize(Collection);
    // TODO: SaveGame에서 Settings를 로드해 복원
}

void UFocusSubsystem::Deinitialize()
{
    // 진행 중인 타이머를 정리
    if (UWorld* World = GetWorld())
    {
        World->GetTimerManager().ClearTimer(FocusTimerHandle);
    }
    Super::Deinitialize();
}

// ── 타이머 시작 ─────────────────────────────────────
void UFocusSubsystem::StartFocus()
{
    if (CurrentState != EFocusState::Idle) return;

    CurrentCycleIndex = 0;
    ConsecutiveCount = 0;
    TransitionTo(EFocusState::Focusing);
}

// ── 일시정지 ────────────────────────────────────────
void UFocusSubsystem::PauseFocus()
{
    if (CurrentState == EFocusState::Idle ||
        CurrentState == EFocusState::Paused) return;

    // 남은 시간을 저장한 뒤 타이머를 해제
    UWorld* World = GetWorld();
    if (!World) return;

    RemainingTime = World->GetTimerManager().GetTimerRemaining(FocusTimerHandle);
    World->GetTimerManager().ClearTimer(FocusTimerHandle);

    StateBeforePause = CurrentState;
    CurrentState = EFocusState::Paused;
}

// ── 재개 ────────────────────────────────────────────
void UFocusSubsystem::ResumeFocus()
{
    if (CurrentState != EFocusState::Paused) return;

    // 저장해 둔 남은 시간으로 타이머를 다시 설정
    CurrentState = StateBeforePause;

    UWorld* World = GetWorld();
    if (!World) return;

    World->GetTimerManager().SetTimer(
        FocusTimerHandle,
        this,
        &UFocusSubsystem::OnTimerExpired,
        RemainingTime,
        false  // 반복 아님 (1회)
    );
}

// ── 강제 중지 ───────────────────────────────────────
void UFocusSubsystem::StopFocus()
{
    if (UWorld* World = GetWorld())
    {
        World->GetTimerManager().ClearTimer(FocusTimerHandle);
    }
    CurrentState = EFocusState::Idle;
    // 중도 중지: XP 미지급. Gold는 이미 틱된 만큼 유지.
}

// ── 스톱워치 시작 ───────────────────────────────────
void UFocusSubsystem::StartStopwatch()
{
    if (SWState != EStopwatchState::Idle) return;

    StopwatchElapsed = 0.f;
    SWState = EStopwatchState::Running;
    // 경과 시간은 Tick()에서 DeltaTime을 누적
}

// ── 스톱워치 중지 ───────────────────────────────────
void UFocusSubsystem::StopStopwatch()
{
    if (SWState == EStopwatchState::Idle) return;

    SWState = EStopwatchState::Idle;

    // 25분 이상이면 보상 브로드캐스트
    if (StopwatchElapsed >= 1500.f)
    {
        OnStopwatchStopped.Broadcast(StopwatchElapsed);
    }
}

// ── 타이머 만료 콜백 ────────────────────────────────
void UFocusSubsystem::OnTimerExpired()
{
    switch (CurrentState)
    {
    case EFocusState::Focusing:
    {
        // 사이클 완료 → 보상 브로드캐스트
        CurrentCycleIndex++;
        ConsecutiveCount++;
        OnFocusCycleCompleted.Broadcast(CurrentCycleIndex, Settings.FocusDuration);

        // 다음 상태 결정
        if (CurrentCycleIndex >= Settings.RoutineCount)
        {
            TransitionTo(EFocusState::LastResting);
        }
        else
        {
            TransitionTo(EFocusState::Resting);
        }
        break;
    }
    case EFocusState::Resting:
        TransitionTo(EFocusState::Focusing);
        break;

    case EFocusState::LastResting:
        if (Settings.bAutoRestart)
        {
            CurrentCycleIndex = 0;
            TransitionTo(EFocusState::Focusing);
        }
        else
        {
            CurrentState = EFocusState::Idle;
        }
        break;

    default:
        break;
    }
}

// ── 상태 전환 + 타이머 시작 ─────────────────────────
void UFocusSubsystem::TransitionTo(EFocusState NewState)
{
    CurrentState = NewState;
    BeginTimerForState(NewState);
}

void UFocusSubsystem::BeginTimerForState(EFocusState State)
{
    float Duration = 0.f;
    switch (State)
    {
    case EFocusState::Focusing:    Duration = Settings.FocusDuration;    break;
    case EFocusState::Resting:     Duration = Settings.RestDuration;     break;
    case EFocusState::LastResting: Duration = Settings.LastRestDuration; break;
    default: return;
    }

    RemainingTime = Duration;

    UWorld* World = GetWorld();
    if (!World) return;

    World->GetTimerManager().SetTimer(
        FocusTimerHandle,
        this,
        &UFocusSubsystem::OnTimerExpired,
        Duration,
        false
    );
}

⚠️
FTimerHandle은 일시정지 시 ClearTimer → 재개 시 남은 시간으로 SetTimer. 엔진의 FTimerManager에는 "일시정지" API가 없습니다. 따라서 일시정지 시 GetTimerRemaining()으로 남은 시간을 읽고 ClearTimer()로 해제한 뒤, 재개 시 그 남은 시간을 SetTimer()에 전달해야 합니다. 이 과정에서 RemainingTime 저장을 빠뜨리면 재개 시 전체 시간이 다시 시작됩니다.
07 — 확장 & 주의

흔한 실수와 권장 패턴

주의해야 할 함정 (Pitfall)

⚠️
Tick 기반 vs FTimerHandle 선택: 타이머 카운트다운에는 FTimerHandle을 사용하세요. Tick은 매 프레임 호출되어 불필요한 오버헤드가 생기고, 프레임 레이트에 따라 정밀도가 달라집니다. 반면 스톱워치의 경과 시간 누적에는 Tick()에서 DeltaTime을 더하는 방식이 적합합니다. 스톱워치는 "만료"라는 개념이 없어 FTimerHandle의 콜백이 필요 없기 때문입니다.
⚠️
일시정지 시 남은 시간 저장을 잊으면 재개가 불가능합니다. PauseFocus()에서 반드시 GetTimerRemaining()으로 남은 시간을 RemainingTime에 저장하세요. 이 값 없이 ClearTimer()만 하면, 재개 시 어디서부터 이어야 하는지 알 수 없어 전체 시간을 처음부터 다시 세게 됩니다.
⚠️
자동 재시작 시 사이클 인덱스 초기화를 빠뜨리면 무한 LastResting에 빠집니다. OnTimerExpired()의 LastResting 분기에서 bAutoRestart가 true일 때, CurrentCycleIndex = 0을 반드시 실행하세요. 초기화하지 않으면 인덱스가 이미 RoutineCount 이상이므로 다음 Focusing 만료 시 바로 다시 LastResting으로 빠져 무한 루프가 됩니다.

Do / Don't

✓ 이렇게

  • 타이머 카운트다운은 FTimerHandle로, 스톱워치 경과 누적은 Tick()으로 분리
  • 일시정지 시 RemainingTime을 저장하고, 재개 시 그 값으로 SetTimer
  • 사이클 완료 알림은 OnFocusCycleCompleted 델리게이트로 브로드캐스트
  • 보상 적립은 EconomySubsystem에 위임 (FocusSubsystem에 금액 계산 로직을 두지 않음)
  • 세션 기록은 RecordSubsystem에 전달 (관심사 분리)

✗ 이건 피하기

  • 타이머를 Tick()으로 구현하면 프레임 레이트 의존성 + 불필요한 CPU 비용 발생
  • ClearTimer 전에 남은 시간을 읽지 않으면 재개 불가
  • FocusSubsystem 안에서 직접 Gold/XP를 변경하면 경제 로직이 분산됨
  • 자동 재시작 시 CurrentCycleIndex를 초기화하지 않으면 무한 루프
  • GetWorld() 반환값을 null 체크 없이 사용하면 에디터 종료 시 크래시
💡

세션 종료 시 RecordSubsystem에 기록을 전달할 때 델리게이트를 사용하면 결합도가 낮아집니다. UFocusSubsystem이 URecordSubsystem의 헤더를 #include하지 않아도, OnFocusCycleCompleted 델리게이트에 RecordSubsystem이 바인딩하면 자동으로 기록이 전달됩니다. 이 패턴은 나중에 "사이클 완료 시 업적 체크" 등 새 기능을 추가할 때도 FocusSubsystem 코드를 건드리지 않고 확장할 수 있게 합니다.
